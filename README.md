# Sales & Revenue Dashboard

A modern, responsive, and performance-optimized sales and revenue dashboard built using React 18, Vite, Tailwind CSS v4, Lucide React, and Recharts.

## Key Features

- **Revenue Overview**: Interactive line charts showing monthly revenue and profit trends.
- **Sales by Category**: Interactive pie chart breaking down sales by product category.
- **Top Products Table**: Paginated list of top performing products with statuses.
- **Dynamic Filters**: Month and category filters that update charts in real time.
- **Responsive Layout**: Designed for seamless display on desktops, tablets, and mobile devices.

## Technology Stack

- **Core**: React 18, React DOM 18, TypeScript
- **Bundler**: Vite
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Charts**: Recharts

## Getting Started

### 1. Install Dependencies

```bash
npm install
```

### 2. Start the Development Server

```bash
npm run dev
```

### 3. Build for Production

```bash
npm run build
```

This project is optimized and ready for zero-configuration Vercel deployment.
